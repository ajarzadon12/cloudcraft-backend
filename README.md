# README

## Requirements

This application currently works with:

* Rails 5.2.2
* MongoDB

## Initial Setup
```
git clone https://github.com/ajarzadon12/cloudcraft-backend.git
cd cloudcraft-backend
bundle install

```

## Usage

Start mongo server
```
[macOs] brew services start mongodb-community@4.0
```

To run the server:
```
rails server -p 4000
```
